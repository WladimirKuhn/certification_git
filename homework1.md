# Подсказка по GIT

## Инициализация пустого репозитория:
```sh
git init
```

## Добавить файл к отслеживанию:
```sh
git add
```
## Добавить коммит:
```sh
git commit -m "Message"
```
## Просмотр истории коммитов
```sh
git log
```
## Просмотр истории коммитов в кратком виде:
```sh
git log --oneline
```
